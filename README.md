## Research

I study racial disparities in financial markets using causal 
inference and machine learning.

### [Persistent Racial Disparities in U.S. Mortgage Approval](https://github.com/Rajveer-code/HMDA-Racial-Disparities-Mortgage-Lending)
42 million applications · 2020–2024 · 5,500 lenders  
Regression discontinuity · Difference-in-differences · DFL decomposition  
Within-lender racial approval penalty: **10.6 percentage points**  
→ *Submitted — Journal of Housing Economics*

### [Heterogeneous Treatment Effects in Mortgage Lending](https://github.com/Rajveer-code/CATE-HMDA-Heterogeneous-Effects)
Extends the above: *for whom is the gap largest?*  
Causal Forests · Double Machine Learning · SHAP attribution  
→ *In progress — arXiv preprint targeted 2026*
